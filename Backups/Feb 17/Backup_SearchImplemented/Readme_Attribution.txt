RichMediaJournal Alpha 2

New Features:
- NewLine Characters: 
	use shift + enter/return to type newline characters directly in the editor
- Some bugfixes

- Load and save profiles use a UUID now internally as well per user.

- Fixed profile creation bugs (some)

- added text formatting such as **bold** *italic* __underline__ --crossout--

- Added inline color formatting.
	Ex. "[Red] R [Green] A [Yellow] I [PastelGreen] N [PastelBlue] B [NeonYellow] O [NeonGreen] W"

- Clickable links: 
	Ex. writing http:www.google.com as part of a message lets you click the link and it will open in your default browser now once sent.
