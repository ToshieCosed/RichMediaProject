Welcome to !!! RichMediaJournalAlpha 3
New Features
- Support Search Bar now in top right of channel chat
	X button will close search
	Enter button will search
		Attributions for a resource used:
		Search Icon:
		Source: https://www.flaticon.com/free-icon/search_18573519?term=search&page=6&position=69&origin=search&related_id=18573519
		Author of search icon: https://www.flaticon.com/authors/ranukumbololab

	-Further Search Features:
		Search will allow filtering channel posts by specific word. It is "whole string" sensitive, so two terms will likely not result
		in messages with one or the other. Only a message containing the whole double term ie "shoe strings" would only return
		messages where the entire string "shoe strings" is mentioned directly. 

-AutoScroll pre-implemented:
	Chat will now auto-scroll to bottom of any selected channel

-Allow Editing Profile Names and Images (independently)
	Sidebar on profile preview has Rename Text box
	Sidebar on profile preview has Update Image Button

RichMediaJournal Alpha 2

New Features:
- NewLine Characters: 
	use shift + enter/return to type newline characters directly in the editor
- Some bugfixes

- Load and save profiles use a UUID now internally as well per user.

- Fixed profile creation bugs (some)

- added text formatting such as **bold** *italic* __underline__ --crossout--

- Added inline color formatting.
	Ex. "[Red] R [Green] A [Yellow] I [PastelGreen] N [PastelBlue] B [NeonYellow] O [NeonGreen] W"

- Clickable links: 
	Ex. writing http:www.google.com as part of a message lets you click the link and it will open in your default browser now once sent.
